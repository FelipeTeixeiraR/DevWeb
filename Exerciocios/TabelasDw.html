Tabelas em HTML

Células de tabela:

é cirado usando "< table >" para definir uma tabela
depois o < tr > para as linhas, e depois o < td > par as células da tabela
Ex:

Ésta é a célula 1 Ésta é a célula 2 Ésta é a célula 3

Linhas da tabela

É usado o < tr > para criar linhas para sua tabela.
Ex:

célula 1 célula 2
célula 3 célula 4
célula 5 célula 6

Cabeçalhos de tabela

Neste é usado o < th > que serve para criar cabeçalhos para sua tabela.
Ex:

Google YouTube Classrom
link Google link YouTube link classrom

Tag HTML < caption >

A Tag Caption serve para adiciona uma descrição para as suas tabelas.
Ex:

Tags de formatação de texto
< b > (negrito) < i > (italico) < del > (taxação)

Tag < colgroup >

O < colgroup > serve par formatar colunas da tabela como mudar a cor e letras.
Ex:

cabeçario 1 cabeçario 2 cabeçario 3
celula 1 celula 2 celula 3
