Fundamentos de HTML

  • Hypertexto: Sistema de navegação que usa links.
  • HTTP: Protocolo para transferência de dados.
  • WWW: World Wide Web, rede global de documentos.

Tags de Formatação

  • <b>: Negrito.
  • <i>: Itálico.
  • <u>: Sublinhado.
  • <strong>: Negrito semântico.
  • <em>: Itálico semântico.

Cabeçalhos e Parágrafos

As tags de <h1> até <h6> são usadas para definir títulos, e a tag <p> define parágrafos.

Imagens e Listas

Imagens são inseridas com <img> e listas com <ul> ou <ol> para listas não ordenadas e ordenadas, respectivamente.

Hyperlinks

Links são criados com a tag <a href="url">Texto do Link</a>.

Tabelas

Tabelas são criadas com <table>, linhas com <tr>, e células com <td> e <th>.

Multimídia

Vídeos e áudios são inseridos com <video> e <audio>.

O que é CSS

CSS é usado para estilizar elementos HTML,
como layout, cores e fontes.

Classes no CSS

Classes são definidas no HTML com class="nome_da_classe" e estilizadas no CSS com .nome_da_classe { propriedade: valor; }.

Estilizar Fontes e Cores

As fontes podem ser estilizadas com font-family, font-size, e color.

Mudar o Tamanho de Imagens

O tamanho de imagens pode ser ajustado com width e height.

Estilizar Elementos Padrões

Elementos como body podem ser estilizados com background-color, font, e outras propriedades CSS.

Seletores no CSS

Seletores podem ser .classe, #id ou o próprio elemento.

Seletores Personalizados

Use pseudo-classes como :hover ou pseudo-elementos como ::before e ::after.

Parallax

O efeito parallax pode ser feito com background-attachment: fixed em uma imagem de fundo.

História do JavaScript

O JavaScript foi criado por Brendan Eich em 1995 para tornar as páginas da web interativas. Inicialmente, era chamado de "Mocha", depois "LiveScript", e finalmente "JavaScript". Hoje, é amplamente utilizado tanto no front-end quanto no back-end de aplicações web.

Como Usar JavaScript no HTML

Você pode inserir JavaScript diretamente dentro de um arquivo HTML usando a tag <script>. Por exemplo:


    <script>
            alert('Olá, mundo!');
    </script>

Esse código exibe uma caixa de alerta quando a página é carregada.

Exemplo de JavaScript Externo

Para manter seu código JavaScript separado, você pode criar um arquivo .js e vinculá-lo ao HTML:

     <script src="meuArquivo.js"></script>

Variáveis e Tipos de Dados

Em JavaScript, você pode declarar variáveis com var, let, ou const. Exemplo:

  
    let nome = 'João';
    const idade = 30;

Funções em JavaScript

Funções são blocos de código reutilizáveis. Exemplo de função que soma dois números:


    function somar(a, b) {
    return a + b;
    }
    console.log(somar(5, 3)); // Saída: 8

Manipulação do DOM

Com JavaScript, você pode manipular os elementos da página HTML, como alterar o texto de um parágrafo:


    document.getElementById('meuParagrafo').innerText = 'Texto alterado';

Eventos em JavaScript

Você pode capturar eventos, como cliques de botões, e reagir a eles. Exemplo:


    document.getElementById('meuBotao').addEventListener('click', function() {
    alert('Botão clicado!');
    });
